From research data to spatial exploration.

This project turns structured event data into a navigable visual interface. It is designed as a compact, public-facing portfolio piece that highlights research, data organisation, and web-based visualisation skills developed through work on organised violence and conflict event data.

01

Structured event data

Events are organised through fields such as date, location, dyad, type of violence, fatalities, and source references.

02

Interactive mapping

The map provides a fast way to inspect spatial patterns, clusters, and concentrations of violence across the dataset.

03

Research portfolio

The website presents a practical demonstration of applied research, data cleaning, geospatial thinking, and front-end design.
